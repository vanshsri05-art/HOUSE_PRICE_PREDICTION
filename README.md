# HOUSE_PRICE_PREDICTION
## 📌 Overview
This is a Machine Learning project that predicts house prices based on features like area, number of bedrooms, and bathrooms using Python and Streamlit. 
## ⚙️ Tech Stack
- Python
- Scikit-learn
- Streamlit
- numpy

## 🚀 How to Run
```bash

streamlit run app.py
